If R is not a member of itself, then its definition entails that it is a member of itself; yet, if it is a member of itself, then it is not a member of itself, since it is the set of all sets that are not members of themselves. The main problem with axiomatic systems is that it is hard to construct a proof; we need intuition. Tableau is a method to prove satisfiability of a set of formulas. It is easy to use and suitable for computer implementation.
